Title : BMSCE STUDENT PROJECT SUBMISSION SYSTEM

Overview

The Website exclusively aims on front-end appearance and provide attractive front-end framework which provides user friendly interface for students and teacher for academic project submission and evaluation.

The index page i.e HOME contains options to login(for students and teacher both), admin login seperately which leads to respective dashboards where students can submit and get results, teachers can evaluate projects and enter marks and admin can maintain the web server(Limited features as no backend is being used).

Technolgies used : HTML, CSS and JavaScript
